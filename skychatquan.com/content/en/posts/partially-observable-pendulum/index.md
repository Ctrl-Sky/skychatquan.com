# Project Overviews
I worked as a Machine Learning Research Assistant under Dr. Stephen Kelly at McMaster University. I worked there from August 2023 - September 2023. Throughout those 4 months, I primarily worked on evolving adaptable control policies using gymnasium’s pendulum task and Distributed Evolutionary Algorithms in Python (DEAP) to conduct experiments on the partially observable pendulum task using genetic
programming. I trained the agent by following Darwin’s Law of Natural Selection, developed and trained multiple generation of AI agents to solve various tasks using reinforcement learning.

My work laid the foundation for other projects now continued by Dr. Stephen Kelly’s
students:
[ https://creativealgorithms-cd4c88.gitlab.io/team/](https://creativealgorithms-cd4c88.gitlab.io/)
